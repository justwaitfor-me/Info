## Exkurs - Experimente mit einem Hexeditor
> Aufgabe 1

    B = 0100 0010
    C = 0100 0011

    00001101 = carrige return (/r)
    00001010 = line feed

    Man kann 256 Zeichen darstellen (2^8).

    UTF-8 Sonderzeichen, wie zum Beispiel [space], werden als '.' dargestellt.
